
# Foobar
(大标题）


## Installation
（小标题）

创建链接 [pip](https://pip.pypa.io/en/stable/)

```bash
三个反引号（```）用于创建代码块（Code Blocks）

pip install foobar
```

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
